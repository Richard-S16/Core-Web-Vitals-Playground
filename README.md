# Core-Web-Vitals-Playground
A web app that analyzes any website’s performance and explains WHY it is slow.

## Features
- **URL Performance Audit:** Real-time data from PageSpeed Insights API.
- **Visual Metric Breakdown:** Interactive charts for LCP, CLS, and TTI.
- **Smart Suggestions:** Automated performance improvements (e.g., preloading, image optimization).
- **JS Bundle Analysis:** Tree-map visualization of library impact.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Visualization:** Recharts / D3
- **Data Fetching:** PageSpeed Insights REST API
- **Optimization:** Web Workers for heavy data processing
